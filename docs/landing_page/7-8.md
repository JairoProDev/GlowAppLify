### 7.3 TRACKING SETUP

**PostHog Events:**

```javascript
// Page view
posthog.capture('$pageview')

// Hero CTA click
posthog.capture('hero_cta_clicked', {
  cta_text: 'Start Your Free Transformation'
})

// Scroll milestones
posthog.capture('scroll_depth', {
  depth_percentage: 25 | 50 | 75 | 100
})

// Demo interaction
posthog.capture('demo_interacted', {
  goal_entered: 'Launch startup'
})

// Testimonial view
posthog.capture('testimonial_viewed', {
  testimonial_id: 'sarah_m'
})

// FAQ expanded
posthog.capture('faq_expanded', {
  question: 'Is this another productivity app?'
})

// Final CTA click
posthog.capture('bottom_cta_clicked')

// Signup initiated
posthog.capture('signup_initiated')

// Signup completed
posthog.identify(user.email, {
  email: user.email,
  signup_source: 'landing_page'
})
posthog.capture('signup_completed')
```

**Session Replay:**
- Enable for 10% of sessions
- Review drop-offs manually
- Find UX issues
